THE GREY CLAN EPISODE ONE: IN CANDLELIGHT
Technical Manual – BGT-WeiDU/EET/BG:EE Edition, v1.9

Requirements, Levels of Compatibility

The installation package of this mod is WeiDU-based. (In common parlance: "This is a WeiDU mod.")

Requirements:

1. A working installation of BGT-WeiDU or Baldur's Gate: Enhanced Edition or Enhanced Edition Trilogy (EET). You don't have to start a new game if certain conditions are met. Please read this.
2. The availability of all, directly or indirectly required original game resources is assumed.
3. The basic functionality of all, directly or indirectly required game resources is assumed as unmodified.

Notes to requirements:
2. The game installation does not have to be unmodified, but removed or renamed resources that are normally present in the original game may cause incorrect mod behaviour and errors. (Example: this edition of the mod does not work with original Baldur's Gate/TotSC or BG1Tutu/EasyTutu, and may not work with any total or partial game conversions.)
3. The game installation does not have to be unmodified, but the basic functionality of resources is required. (Example: replaced original travel regions that lead to different areas than in the original game may cause incorrect mod behaviour and errors.)

Compatibility on technical level:

This mod is compatible with all other standard WeiDU-based mods if they meet the above requirements.
A valid filename prefix is used for all installed resource files of this mod. The prefix is registered in the Community Filenames Prefix Reservations database.
This mod installs new resources.
This mod does not overwrite any existing game resources.
This mod affects (modifies or extends) existing game resources on data (content) level.

When this mod is uninstalled, all changes except TLK entries are safely removed except where cached and/or saved by the game or player. Note: the remaining TLK entries are unused, therefore they won't have any practical effect on the game.

Compatibility on game level:

This mod partically affects existing game resources on data (content) level, and therefore (direct or indirect) object-level interaction is possible in the game with mods that affect the same resources. See the list of resources affected on data level to see which resource attributes are modified and extended by this mod.

A valid prefix is used for all variables of this mod. The prefix is registered in the Community Filenames Prefix Reservations database.

For more information, please see the levels of coupling.

Levels of Coupling

Resource level:

  1. File level:
    No (original game or third-party mod) files are overwritten or deleted.
    Original game files are required.
    No third-party mod files are required or used optionally.
  2. Data level:
    Original game resources are modified.
    Original game resources are extended.
    No third-party mod files are modified or extended.

Game level:

  1. Variable level:
    Values of (original game or third-party mod) variables are not modified.
    Values of (original game or third-party mod) variables are not read and used.
  2. Object level:
    No (original game or third party mod) actors or items are checked and used by the scripts of this mod.
    Original game area(s) are referenced and used by the script(s) of this mod: AR6700 is unavailable from a certain point of the plot of this mod until the end of the plot of this mod. Changes made by the original game or by third-party mods on AR6700 are not blocked, delayed or affected in any other way.
    No third-party mod areas are referenced and used by the scripts of this mod.
    Except what specified above, no other (original game or third party mod) objects are checked or used by the scripts of this mod.
    Known interactions:
    • Certain (unique or common) actors of this mod may attack any game actors of class (class.ids) INNOCENT, FLAMING_FIST and FIGHTER. Areas with high risk: AR7600, AR7200. No other risky areas are known for this risk.
    • Certain (unique or common) actors of this mod may attack any game actors of EA (ea.ids) ENEMY, except if class (class.ids) is INNOCENT or FLAMING_FIST. Areas with high risk: AR7600, AR7300, AR7200, AR7263, AR7264. No other risky areas are known for this risk.
    Since this mod affects (modifies or extends) existing game resources on data level, direct or indirect object-level interaction is possible in the game with mods that affect the same resources in certain ways.
  3. Time and event level:
    The plot of this mod is controlled by nothing else than the logical order of events triggered by (player or other in-game) actions. No timers are used.

Summary:
The coupling between the original game and this mod is considered to be: LOOSE.
The coupling between the original game and other mods is assumed to be: NEGLIGIBLE.

Cohesion

The cohesion of this mod is: VERY BIG.
